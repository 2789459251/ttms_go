本项目为基于golang语言完成TTMS影院管理系统

1.数据存储：

    Redis：用于缓存和快速读取数据，例如用户会话、影片信息等。
    MySQL：用于持久化存储数据，例如用户信息、订单信息等。
2.框架：

    Gin：Golang的Web框架，用于处理HTTP请求和构建API接口。
    ORM库：

    GORM：Golang的ORM库，用于与MySQL数据库进行交互，执行数据库操作和管理数据模型。
3. 中间件：

    JWT：用于身份认证和授权，通过生成和验证JSON Web Tokens来实现用户身份验证和访问控制。
4.工具：

    Kafka：用于实现消息队列和异步通信，支持高吞吐量的分布式消息传递。
    Viper：用于读取和解析配置文件，可以方便地管理应用程序的配置参数。
    Swagger：用于生成API文档和提供交互式API界面，方便开发者查看和测试API接口。
5.存储：

    青牛云对象存储：选择用于存储和管理影院系统的文件资源，例如影片海报、用户上传的图片等。
